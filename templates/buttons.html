Appendix:

  Modifier Class Assign To Details
.btn * Applies default button style to element.
.btn--dismiss .btn Applies secondary 'dismiss' button style to a button.
.btn--brand .btn Applies secondary 'brand' button style to a button.
.btn--progress .btn Applies secondary 'progress' button style to a button.
.btn--white .btn Applies secondary 'white' button style to a button.
.btn--link .btn Applies tertiary 'link' button style to a button.
.btn--disabled .btn Applies disabled appearance to a button.
.btn--block .btn Forces button to be 100% wide.
