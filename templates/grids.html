Basic Grid Template:

  Modifier Class Assign To Details
  .grid * Wraps around grid columns to establish grid formating.
  .grid__col .grid > * Used to define a grid column. By default, the column is set to be 100% wide.
  .grid__col--1of12 .grid__col Set grid column to 1/12 wide.
  .grid__col--2of12 .grid__col Set grid column to 2/12 wide.
  .grid__col--3of12 .grid__col Set grid column to 3/12 wide.
  .grid__col--4of12 .grid__col Set grid column to 4/12 wide.
  .grid__col--5of12 .grid__col Set grid column to 5/12 wide.
  .grid__col--6of12 .grid__col Set grid column to 6/12 wide.
  .grid__col--7of12 .grid__col Set grid column to 7/12 wide.
  .grid__col--8of12 .grid__col Set grid column to 8/12 wide.
  .grid__col--9of12 .grid__col Set grid column to 9/12 wide.
  .grid__col--10of12 .grid__col Set grid column to 10/12 wide.
  .grid__col--11of12 .grid__col Set grid column to 11/12 wide.
  .grid__col--12of12 .grid__col Set grid column to 12/12 wide. (same as .grid__col)
 
 
 
 
 
 
 
 
 
 
 
 
 

Responsive Grid Template:

  Modifier Class Assign To Details
  --medium .grid__col Defines the size of a grid column for medium sized devices and larger.
  --large .grid__col Defines the size of a grid column for large sized devices and larger.
  --xlarge .grid__col Defines the size of a grid column for extra large sized devices.
