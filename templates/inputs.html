Appendix:

  Modifier Class Assign To Details
.input <input /> Applies input text field styles and makes field 100% wide.
.input--number .input Removes default browser increments/arrows from the field.
.input--textarea .input Applies a defined height to a text field.
.input--micro .input Applies 100px width to input field.
.input--sm .input Applies 30% width to input field.
.input--med .input Applies 48% width to input field.
  .input--hasError .input Applies error styling to the input field.
