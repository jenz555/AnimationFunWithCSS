Appendix:

  Modifier Class Assign To Details
.vList * Formats all child elements to appear as a vertical list.
.vList--piped .vList Applies a line separator between all vertical list items.
.vList--tight .vList Reduces vertical spacing between all vertical list items.
.hList * Formats all child elements to appear as a horizontal list.
.hList--center .hList Centers all child elements in a horizontal list.
.hList--right .hList Right aligns all child elements in a horizontal list.
.hList--justify .hList Justifies all child elements in a horizontal list.
.hList--top .hList Top aligns all child elements in a horizontal list.
.hList--middle .hList Middle aligns all child elements in a horizontal list.
.hList--bottom .hList Bottom aligns all child elements in a horizontal list.
.hList--piped .hList Applies a line separator between all horizontal list items.
.hList--toVList .hList Formats all list items into a horizontal list on mobile devices but a vertical list on larger devices.
.vList--toHList .vList Formats all list items into a vertical list on mobile devices but a horizontal list on larger devices.
.uList * Prefixes the default browser bullet to each list item.
.oList * Prefixes the default browser number sequence to each list item.
