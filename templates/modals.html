Appendix:

Option Type Default Details
overlay boolean true If modal has class .dialogNonModal, this value automatically will be set to false.
blockOverlayClick boolean false If set to true, click events attached to the overlay will be canceled.
TertiaryBtn boolean true If set to false, the dialog will not have a Tertiary button.
activeClass string dialogModal--active If modal has class .dialogNonModal, this value will automatically be set to dialogNonModal--active
Callback Details
onShow Triggers callback when modal is opened.
onHide Triggers callback when modal is Tertiaryd.
