Allstate© User Interface Toolkit Tables Template

Appendix:

  Modifier Class Assign To Details
.table--container .table Wraps around a table and adds a horizontal scrollbar for mobile devices.
.table <table> Applies branded table styles.
.table--secondary .table Adds secondary table styles.
.table--striped .table Adds stripes to every even number of rows.
  .table--fixed .table Justifies the size of table cells to be the same width.
.table--bordered .table Adds vertical borders to all table cells.
.table--emptyBG .table Adds styling on empty table cells.
.table--responsive .table Formats table to appear as a list on mobile devices.
.table__row--isActionable .table tr Adds styling to a table row indicating that it contains an actionable item.
