Allstate© User Interface Toolkit Tab Template

Notes:

Tab is a common container designed to display as a tab control across all screen sizes.

All display toggle containers (tabs, accordions, tabcordions) are built on the toggle-core.plugin script which manages display state and emits events. Toggle containers work on the concept of triggers (denoted by data-trigger) and targets (denoted by data-target).

The tab script layers on top of toggle-core to add keyboard navigation and tab-specific accessibility. (Arrow keys navigate between individual tabs).

css:

Modifier Class Assign To Details
.tabs * Wraps around tab content and manages its layout.
.tabs--colAlign .tabs Displays the tab content in column format.
.tabs--colAlign--reverse .tabs Use to reverse the display order of column formatted tabs
.tabs--rowAlign .tabs Displays the tab content in row format.
.tabs--rowAlign--reverse .tabs Use to reverse the display order of row formatted tabs
.tabs--justified .tabs Ensures that tab items fill 100% width and each occupies the same individual width. Use in conjunction with tabs--rowAlign only.
tabs__hd * Wrapper for tab items.
.tabs__hd__tab * Defines an individual tab item.
.tabs__bd * wrapper for tab content panels.
.tabs__bd__content * Wrapper container for tab content.
