Appendix:

  Modifier Class Assign To Details
default * By default, all text renders at 1rem / 16px.
.txt--bold .txt Apply bolded font-weight to text.
.txt--semibold .txt Apply semi-bolded font-weight to text.
.txt--underline .txt Apply underline to text.
.txt--italic .txt Apply italic font style to text.
.txt--strike .txt Apply strikethrough styling to text.
.txt--xs .txt Resize text to .75rem / 12px.
.txt--sm .txt Resize text to .875rem / 14px.
.txt--lg .txt Resize text to 1.125rem / 18px.
.txt--brand .txt Changes text to be the "@brand" color.
.txt--dusty--x4 .txt Changes text to be the "@dusty--x4" color.
.txt--dusty--x5 .txt Changes text to be the "@dusty--x5" color.
.txt--success .txt Changes text to be the "@tender" color.
.txt--inverse .txt Changes text to be white.
