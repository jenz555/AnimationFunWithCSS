Appendix:

Option Type Default Details
Scheduler-Style Timepicker
onload boolean false If set to true, timepicker will open on page load.
showSetToNow boolean false If set to true, a button will appear below the timepicker input allowing the user to set the value as the current time.
modifierClass string '' Can be used to set a modifier class on the parent container.
defaultTimeZone string Timezone specified by computer. Default time zone is based off of user's computer.
timezones array [] Can be overrriden with any array of objects containing time zone key/value pair.
Eg. [{ 'EST': 'Eastern Standard Time' }]
minuteIncrements number 5 Can be overridden to set the minute increments in the timepicker. Can be set to 1, 5, 15, 30 or 45.
Dropdown-Style Timepicker
modifierClass null string Appends a modifier class to the timepicker wrapper.
selectedStartTime null string Default selected start time for a time range.
selectedEndTime null string Default selected end time for a time range.
blockOutTimes array [] Array of times that will be excluded from timepicker.
startTime string 12:00am Default start time - first selectable option.
endTime string 11:00pm Default end time - last selectable option.
interval number 60 Can be set to show 15, 30 or 60 minute time intervals.
step boolean false If set to true, two TimePicker dropdowns will be displayed.
Callback Details
Scheduler-Style Timepicker
onShow Triggers a callback when the timepicker is opened.
onHide Triggers a callback when the timepicker is closed.
