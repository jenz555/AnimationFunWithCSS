Appendix:

  Modifier Class Assign To Details
  .toggle * Manages the format of the toggle buttons.
  .toggle__btn .toggle > * Defines a button within the button group.
  .toggle_radio <input type="radio" /> Applies button styling to a radio button.
  .toggle__label <label> Applies styling to a radio button's associated label.
  .toggle--dusty * Manages the format of the dusty button group.
  .toggle__btn--dusty .toggle--dusty > * Manages the format of the dusty toggle buttons.
  .toggle_radio--dusty <input type="radio" /> Applies dusty button styling to a radio button.
  .toggle__label--dusty <label> Applies dusty styling to a radio button's associated label.
