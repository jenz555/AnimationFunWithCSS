Appendix:

Modifier Class Assign To Details
.tooltip * Wraps around tooltip content.
.tooltip--toaster * Wraps around tooltip content. Applies toaster styles to tooltip.
Option Type Default Details
modifierClass string null Adds a modifier class to the tooltip content container.
delay number 0 Sets the delay for hiding the tooltip on mouseleave.
onclick boolean false If set to true, the tooltip will be triggered on click instead of hover.
Callback Details
onShow Callback to be triggered when tooltip is shown.
onHide Callback to be triggered when tooltip is hidden.
